Table showing URL Requests

Returns URL HTTP Method Query Parameters Requests Response
List of Books /books GET - None JSON containing data of all the books
Book Information /books/:Title GET :title None JSON object containing data as displayed: 
                                
{
    title: '...',
    synopsis: '...',
    genre: {
        name: '...',
        description: '...'
    },
    author: {
        name: '...',
        bio: '...',
        birth: '...',
        death: '...'
    },
    imageURL: '...'
}
Genre /genre/:name GET :name None JSON object containing data as displayed: 
                                
{
    genre: {
        name: '...',
        description: '...'
    }
}
Author /author/:name GET :name None JSON object containing data as displayed: 
                                
{
    author: {
        name: '...',
        bio: '...',
        birth: '...',
        death: '...'
    }
}
Adding a new user /users POST - None JSON object containing data of the new user. Example: 
								
{
	Username: '...',
  Password: '...',
  Email: '...',
  Birthday: MMDDYYYY,
  FavoriteBooks: [...]
}
Adding a new user /users/:username POST :username None JSON object containing the updated info of the current user. Example: 
								
{
	Username: '...',
  Password: '...',
  Email: '...',
  Birthday: MMDDYYYY,
  FavoriteBooks: [...]
}
Adding a book to a user's favorite list /users/:username/books/:bookID POST :bookID None Favorite Books: ['...', '...']
Removing a book to a current user's list /users/:username/books/:bookID DELETE :bookID None Favorite Books: ['...']
Removing a current user /users/:Username DELETE :Username None Username was deleted.
